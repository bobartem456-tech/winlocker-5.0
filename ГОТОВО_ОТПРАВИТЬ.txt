"Привет! Держи бота для управления компьютером через Telegram.

📋 ЧТО СДЕЛАТЬ:
1. Распакуй все файлы которые я прислал в ОДНУ папку
2. Создай своего бота в Telegram через @BotFather
   - Отправь /newbot
   - Придумай имя и username
   - Скопируй токен
3. Узнай свой ID через @userinfobot
   - Отправь любое сообщение
   - Скопируй Id (число)
4. Открой файл .env в блокноте
   - Вставь токен вместо 'ВСТАВЬ_СЮДА_ТОКЕН_ОТ_BOTFATHER'
   - Вставь свой ID вместо 'ВСТАВЬ_СЮДА_СВОЙ_ID'
   - Сохрани
5. Запусти MainBot.exe
6. Открой своего бота в Telegram
7. Нажми /start

ВСЁ! БОТ ГОТОВ! 🚀

Полная инструкция в файле ДЛЯ_ДРУГА.txt"
