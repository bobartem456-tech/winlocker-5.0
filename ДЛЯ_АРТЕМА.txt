1️⃣ ЗАРЕГИСТРИРУЙСЯ НА GITHUB:

2️⃣ СОЗДАЙ РЕПОЗИТОРИЙ:
   
   - Зайди на https://github.com
   - Нажми "+" → "New repository"
   - Name: winlocker-5.0
   - Description: Система удаленного администрирования
   - Public или Private (как хочешь)
   - ✅ Initialize with README
   - Create repository

3️⃣ УСТАНОВИ GIT:
   
   Скачай: https://git-scm.com/download/win
   Установи (все опции по умолчанию)

4️⃣ ПОДГОТОВЬ ПРОЕКТ:
   
   Git БУДЕТ игнорировать (уже настроено в .gitignore):
   ❌ .env (с токенами)
   ❌ *.log
   ❌ __pycache__/
   ❌ dist/, build/
   
   Git БУДЕТ коммитить:
   ✅ Все .py файлы
   ✅ .env.example (без токенов!)
   ✅ Документацию
   ✅ requirements.txt

5️⃣ ОТПРАВЬ НА GITHUB:
   
   Открой Git Bash в папке проекта:
   
   ```bash
   git init
   git add .
   git commit -m "WinLocker 5.0 v16.1 - Security Update"
   git remote add origin https://github.com/ТВОЙ_USERNAME/winlocker-5.0.git
   git push -u origin master
   ```

6️⃣ ОТПРАВЬ ССЫЛКУ АРСЕНИЮ:
   
   Ссылка будет вида:
   https://github.com/ТВОЙ_USERNAME/winlocker-5.0
   
   Напиши Арсению:
   "Привет! Вот проект WinLocker 5.0:
    https://github.com/ТВОЙ_USERNAME/winlocker-5.0
    
    Для запуска:
    1. git clone https://github.com/ТВОЙ_USERNAME/winlocker-5.0.git
    2. cd winlocker-5.0
    3. copy .env.example .env
    4. Вставь свои токен и ID в .env
    5. pip install -r requirements.txt
    6. python main_bot.py"
